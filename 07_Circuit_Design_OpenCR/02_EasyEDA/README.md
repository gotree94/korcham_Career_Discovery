# 2일 과정: EasyEDA를 활용한 OpenCR 회로 설계 (총 14시간)

---

## EasyEDA 개요

| 항목 | 내용 |
|------|------|
| **종류** | 무료 웹 기반 EDA (Electronic Design Automation) |
| **설치** | **불필요** — 브라우저만 있으면 즉시 사용 가능 |
| **플랫폼** | 클라우드 EDA (모든 작업이 서버에 저장) |
| **공식 사이트** | [https://easyeda.com](https://easyeda.com) |
| **제공사** | LCSC / JLCPCB (중국) — 부품 구매/제조와 연계 |

### EasyEDA의 장점

1. **설치 불필요** — Chrome, Edge, Firefox 등 최신 브라우저에서 바로 실행
2. **팀 협업** — 실시간 공유 편집, 댓글, 버전 관리
3. **LCSC 부품 라이브러리** — 100만+ 부품 검색 → 심볼 + 풋프린트 + 3D 모델 + 가격 한 번에 로드
4. **JLCPCB 직접 연계** — 설계 완료 → Gerber → JLCPCB 주문까지 원스톱
5. **크로스 플랫폼** — Windows / macOS / Linux 구분 없음
6. **무료** — 제한 없는 Private 프로젝트, 기본 기능 모두 무료

### OpenCR 참조 보드

- **MCU**: STM32F746ZGT6 (ARM Cortex-M7 @ 216MHz, LQFP-144)
- **IMU**: ICM-20648 (6축 자이로/가속도 센서)
- **전원**: LM5175PWPR (Buck-Boost) + XL4005 (DC-DC) + LDO x2
- **모터 드라이버**: A3906SESTR-T x2 (듀얼 모터)
- **통신**: CAN (TJF1051T/3), RS-485 (MAX3443ECSA+), USB OTG
- **크기**: 105mm × 75mm, 4층 PCB

## 과정 구성

| 일차 | 주제 | 시간 | 세부 내용 |
|------|------|------|-----------|
| **Day 1** | **회로도 (Schematic)** | **7h** | 프로젝트 생성 → 부품 검색 → 전원부 → MCU → 모터/통신 → ERC |
| **Day 2** | **PCB 아트웍 (Layout)** | **7h** | 변환 → 배치 → 배선 → DRC → 거버 출력 → 3D 검토 |

> 각 Step은 **실습 시간 기준**입니다. 설명과 함께 진행하며, 중간중간 질문/토론 시간을 포함합니다.

---

## Day 1: 회로도 (Schematic) — 7시간

| Step | 내용 | 시간 |
|------|------|------|
| **1** | EasyEDA 시작 및 프로젝트 생성 | 40min |
| **2** | 부품 라이브러리 및 심볼 추가 | 1h 20min |
| **3** | 전원부 회로도 작성 | 1h 30min |
| **4** | MCU STM32F746 주변회로 | 1h 30min |
| **5** | 모터 드라이버 + 통신 인터페이스 | 1h |
| **6** | 커넥터 + ERC 검증 | 1h |

## Day 2: PCB 아트웍 (Layout) — 7시간

| Step | 내용 | 시간 |
|------|------|------|
| **7** | PCB 변환 및 레이어 설정 | 1h 30min |
| **8** | 부품 배치 (Placement) | 1h 30min |
| **9** | PCB 배선 (Routing) | 1h 30min |
| **10** | DRC + 거버 출력 | 1h |
| **11** | 3D 시각화 + 설계 검토 | 1h |
| **12** | 최종 발표 + 리뷰 | 30min |

## 사전 준비

1. **브라우저**: Chrome 또는 Edge 최신 버전
2. **계정**: [EasyEDA](https://easyeda.com) 가입 (Google 계정 연동 권장)
3. **참고 자료**: OpenCR 회로도 PDF (원본), OpenCR 포토 이미지
4. **지식**: 기초 전자공학 (저항, 캡, 전원, MCU 개념)
