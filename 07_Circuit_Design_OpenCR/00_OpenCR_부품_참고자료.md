# OpenCR 부품 참고 자료 (4개 EDA 도구 공통)

> OpenCR REV H 기준 주요 부품 목록. 각 EDA 도구 실습에서 동일한 부품을 사용한다.

## 주요 IC

| refdes | 부품번호 | 패키지 | 설명 |
|--------|---------|--------|------|
| U3 | STM32F746ZGT6 | LQFP-144 | MCU, ARM Cortex-M7, 216MHz |
| U4 | LM5175PWPR | HTSSOP-28 | 벅-부스트 컨트롤러 |
| U5 | XL4005 | TO-263-5 | DC-DC 스텝다운 |
| U6 | IL1117-5.0ET | SOT-223 | 5V LDO |
| U23 | AZ1117H-3.3TRG1 | SOT-223 | 3.3V LDO |
| U7, U14 | A3906SESTR-T | QFN-20 | 모터 드라이버 |
| U9 | TJF1051T/3 | SOP-8 | CAN 트랜시버 |
| U11 | MAX3443ECSA+ | SOP-8 | RS-485 트랜시버 |
| U16 | ICM-20648 / MPU-9250 | QFN-24 | 6축/9축 IMU |
| U21 | STMPS2141STR | SOT-23-5 | USB 전원 스위치 |
| U17 | EMIF02-USB03F2 | FlipChip-8 | USB EMI 필터 |
| F1 | 0453010 | SMD | 퓨즈, 10A 125V |

| J15, J16, J17 | B4B-EH-A (JST) | 4 | RS-485 DYNAMIXEL |

| 항목 | 대표값 | 패키지 |

| Resistor 1k | 1kΩ / 1% | R1608 (0603) |

## 전원부 구성

```
입력 5~24V
  │
  ├─ F1 (0453010, 10A) ─┬─ LM5175PWPR ── 12V @ 4.5A
  │                     └─ XL4005 ────── 5V @ 4A
  │                                      │
  │                                      └─ AZ1117H-3.3 ── 3.3V @ 800mA
  └─ IL1117-5.0 ── 5V (USB)
```
