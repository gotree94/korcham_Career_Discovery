# 자율주행하드웨어개발 커리큘럼 개요

## 과정 개요

| 항목 | 내용 |
|------|------|
| 과정명 | 자율주행하드웨어개발 (직업계고 채용연계형 직무교육) |
| 교육기간 | 2026.08.03(월) ~ 2026.08.14(금), 70h |
| 교육대상 | 직업계고 1~2학년 재학생 |
| 교육목적 | 로봇 산업 전 분야(기구·HW·FW·SW·AI)를 경험하며 진로적합성 및 직무역량 조기 형성 |
| 취득역량 | 진로적합성 판단, 자율주행 하드웨어/소프트웨어 기초 이해, Full-stack 스캔 경험 |

## 활용 장비

| 구성 요소 | 사양 |
|-----------|------|
| 메인 컴퓨터 | NVIDIA Jetson AGX Orin (Orin NX 등) |
| 구동계 | 옴니휠 3개 + 각각 BLDC/DC 모터 + 모터 드라이버 |
| 거리 센서 | 초음파 센서 HC-SR04 x 3 |
| 근접 센서 | IR 근접센서 x 3 |
| 환경 센서 | 습도 센서, 화재 감지 센서 |
| 위치 인식 | LiDAR (RPLiDAR 또는 YDLiDAR) |
| 출력 장치 | LCD 디스플레이 |
| 입력 장치 | 무선 키보드 / 마우스 |
| 전원 | 배터리 팩 |
| S/W 플랫폼 | Ubuntu + ROS2 Humble + Python/C++ |
| 설계 도구 | EasyEDA (회로), FreeCAD (기구), 오실로스코프/로직 애널라이저 |

## 교육 철학

> **"100% 이해하지 못해도 괜찮다. 경험했다는 사실이 중요하다."**
> **"로봇은 기구·HW·FW·SW·AI가 함께 움직인다. 그 전부를 맛보자."**

- 직업계고 1~2학년 대상, 전문가 양성이 아닌 **진로 탐색**이 목적
- Full-stack 스캔 방식: 4개 분야(기구·HW·FW·SW/AI)를 짧게 순회하며 경험
- 이론은 시스템이 "왜" 동작하는지 직관적으로 이해하는 수준
- 실습은 "직접 연결하고, 읽고, 움직여보는" 경험 중심 (실습 70%, 이론 30%)
- 모든 실습은 팀 단위 협업으로 진행
- 각 분야 마무리에는 "기억에 남는 한 가지"를 남기는 체험 실습 포함
- 실습 결과물이 최종 통합 프로젝트의 재료가 되는 구조

## 학습 구조 (Full-stack 스캔 방식)

```
Phase 1: Full-stack Domain Scan (8일, 48h)
┌─────────────────────────────────────────────────────┐
│  HW 분야 (3일)          기구 분야 (2일)              │
│  1일차: EasyEDA 회로설계   4일차: FreeCAD 모델링     │
│  2일차: 신호측정+데이터시트 5일차: FEA+3D프린터     │
│  3일차: GPIO 제어                                   │
├─────────────────────────────────────────────────────┤
│  FW 분야 (1일)          SW·AI 분야 (2일)             │
│  6일차: PWM+모터제어     7일차: Linux+GPU           │
│                          8일차: AI+로봇산업현황     │
└─────────────────────────────────────────────────────┘

Phase 2: 통합 및 프로젝트 (2일, 12h)
  9일차: Domain 결과물 → ROS2 통합 실습
 10일차: 자율주행 통합 구현 + 테스트

Phase 3: 콘테스트 및 발표 (2일, 8h)
 11일차: 주행 콘테스트
 12일차: 발표 자료 제작 + 최종 발표 + 수료
```

## 평가 방식

| 구분 | 비중 | 내용 |
|------|------|------|
| 진단평가 | - | 교육 전 진로적성 진단 (참고용) |
| 도메인 실습 | 40% | 각 분야별 마무리 실습 결과물 (8회) |
| 통합 프로젝트 | 40% | 주행 콘테스트 결과 + 통합 구현 완성도 |
| 참여도 | 20% | 출석, 팀 기여도, 태도 |
| 취업지도 | 별도 | 교육생 80% 이상 취업 상담 이수 |
