# 미니프로젝트 가이드

## 프로젝트 개요

자율주행 통합 기능 구현 및 주행 콘테스트

| 항목 | 내용 |
|------|------|
| 기간 | 6~10일차 (총 30h, 통합 구현 + 테스트) |
| 팀 구성 | 4~5인 x 5팀 |
| 평가 | 주행 콘테스트 50% + 발표 50% |

---

## 주행 콘테스트 규칙

### 코스 구성

```
[시작] → [장애물구간] → [화재감지존] → [곡선주로] → [목적지]
```

- 직사각형 코스 (약 3m x 5m)
- 장애물 3~4개 배치 (박스, 콘 등)
- 화재 시뮬레이션 지점 1개 (LED 또는 불빛)
- 곡선 구간 포함

### 평가 기준 (100점)

| 항목 | 배점 | 세부 기준 |
|------|------|----------|
| 코스 완주 | 40점 | 제한시간(5분) 내 완주 (완주 실패 시 0점) |
| 장애물 회피 | 20점 | 장애물 접촉 1회당 -5점 |
| 화재 감지 | 20점 | 화재지점에서 30cm 이내 정지 + LCD 경고 표시 |
| 주행 안정성 | 20점 | 급정지/급회전 없이 부드러운 주행 |

### 감점 요소
- 장애물 접촉: -5점/회
- 코스 이탈: -10점/회
- 화재 미감지: -20점
- 시간 초과: 30초 초과 시 -5점/30초

---

## 필수 구현 기능

### 1. 장애물 회피
- LiDAR + 초음파 + IR 센서 융합
- 전방 40cm 이내 장애물 감지 시 회피 동작
- 회피 후 원래 경로로 복귀

### 2. 화재 감지 및 대응
- 불꽃 센서로 화재 감지
- 즉시 정지 + LCD에 "FIRE!" 표시
- 부저 경고 (선택)
- 3초 후 재주행 (또는 정지 유지 → 규칙에 따라)

### 3. LCD 상태 표시
- 실시간 주행 상태 표시
  - 1라인: 현재 동작 (DRIVING / AVOID / STOP / FIRE!)
  - 2라인: 속도 + 거리 정보

### 4. 옴니휠 정밀 제어
- 부드러운 가속/감속
- 정확한 회전각 제어
- 대각선 이동 가능

## 팀 구성 및 역할

| 역할 | 담당 | 주요 업무 |
|------|------|----------|
| **HW 리드** | 하드웨어 연결 및 점검 | 센서/모터/LiDAR 연결, 배선 관리, 전원 관리 |
| **SW 리드** | 통합 코드 작성 | ROS2 노드 구성, 센서-모터 통합, 메인 루프 |
| **알고리즘** | 주행 로직 개발 | 장애물 회피, 경로 추종, 화재 대응 로직 |
| **UI/발표** | LCD + 발표 준비 | LCD 표시, 테스트 기록, 발표자료 제작 |
| **테스터** | 통합 테스트 및 디버깅 | 시나리오 테스트, 버그 리포트, 캘리브레이션 |

> 인원이 부족한 팀은 SW 리드 + 알고리즘, HW 리드 + 테스터 병합 가능

- 단위 테스트 및 튜닝

**산출물**: 장애물 회피 노드, 주행 테스트 영상

### 8일차: 화재 감지 + 안전 시스템 + 통합

- LCD 상태 표시 노드 구현
- 전체 시스템 1차 통합

**산출물**: 통합 시스템 1차 버전, 화재 대응 테스트 결과

### 9일차: 통합 주행 테스트
- 전체 코스 반복 주행 테스트
- 장애물/화재 시나리오별 테스트
- 파라미터 튜닝 (게인값, 속도, 감지 거리)

**산출물**: 주행 테스트 로그, 튜닝 결과

### 10일차: 최종 튜닝 + 예행 주행
- 최종 캘리브레이션
- 콘테스트 코스 예행 주행
- 발표자료 준비 시작

**산출물**: 최종 코드, 예행 주행 결과

### 메인 통합 노드 예시

```python
#!/usr/bin/env python3
import rclpy
from rclpy.node import Node
from std_msgs.msg import Float32MultiArray, String

class RobotMain(Node):
    def __init__(self):
        super().__init__('robot_main')
        
        # 상태 변수
        self.distance = 999.0
        self.ir_status = [1, 1, 1]
        self.flame_detected = False
        self.state = 'IDLE'  # IDLE, DRIVING, AVOID, FIRE_STOP
        
        # 구독
        self.create_subscription(Float32MultiArray, 'sensor_data', self.sensor_cb, 10)
        
        # 발행
        self.cmd_pub = self.create_publisher(String, 'cmd_vel_custom', 10)
        self.lcd_pub = self.create_publisher(String, 'lcd_text', 10)
        
        # 메인 루프
        self.timer = self.create_timer(0.1, self.main_loop)
    
    def sensor_cb(self, msg):
        self.distance = msg.data[0]
        self.ir_status = msg.data[1:4]
        self.flame_detected = msg.data[4] == 1.0
    
    def main_loop(self):
        if self.flame_detected:
            self.state = 'FIRE_STOP'
            self.handle_fire()
        elif self.distance < 30:
            self.state = 'AVOID'
            self.handle_avoid()
        else:
            self.state = 'DRIVING'
            self.handle_drive()
        
        # LCD 업데이트
        lcd_msg = String()
        lcd_msg.data = f"STATE:{self.state}"
        self.lcd_pub.publish(lcd_msg)
    
    def handle_drive(self):
        self.get_logger().info('전진 주행')
        # omni_drive(0, 50, 0)
    
    def handle_avoid(self):
        self.get_logger().info('장애물 회피')
        # omni_drive(0, 0, 50)  # 회전
        # time.sleep(0.5)
        # omni_drive(50, 30, 0)  # 대각선 이동
    
    def handle_fire(self):
        self.get_logger().info('🔥 화재 감지! 비상 정지!')
        # omni_drive(0, 0, 0)

def main():
    rclpy.init()
    node = RobotMain()
    rclpy.spin(node)
```

## 발표자료 가이드

### 필수 포함 내용
1. **팀 소개** (1슬라이드)
   - 팀명, 팀원, 역할
2. **프로젝트 개요** (1~2슬라이드)
   - 사용 장비, 시스템 아키텍처
   - 하드웨어 구성 사진
3. **구현 기능** (2~3슬라이드)
   - 장애물 회피 알고리즘 설명
   - 화재 감지 및 대응 로직
   - 옴니휠 제어 방식
4. **시연 영상** (1~2분)
   - 실제 주행하는 모습
   - 장애물 회피/화재 감지 장면 포함
5. **트러블슈팅** (1슬라이드)
   - 겪었던 문제와 해결 방법
   - 배운 점
6. **소감 및 향후 계획** (1슬라이드)
   - 개인별 소감 (팀원 각각 1문장)
   - 더 발전시키고 싶은 부분

### 발표 팁
- 10분 발표, 5분 Q&A
- 코드 설명보다는 **동작과 결과**를 중심으로
- 실패한 부분도 솔직하게 공유
- 시연 영상은 필수 (실시간 시연은 리스크 있음)
