## 3. 보고서 작성 실습 (30분)

### 팀 작업

```
1. 위 템플릿을 복사하여 각 팀의 내용 작성
2. FreeCAD 스크린샷 캡처:
   - 최종 조립 이미지
   - FEA 결과 비교 (전/후)
   - 위상 최적화 비교
   - 애니메이션 장면
3. 수치 데이터 채우기
4. PDF 또는 마크다운 파일로 저장
```

### 스크린샷 캡처 방법

```
1. FreeCAD 3D 뷰를 원하는 각도로 설정
2. Tools → Save Picture → Resolution 설정
   → "fea_result.png" 저장
   
또는 Windows 캡처:
   Windows + Shift + S (부분 캡처)
```

## 4. 최종 발표 (30분)

### 발표 형식

| 항목 | 내용 |
|------|------|
| 시간 | 팀당 7분 발표 + 3분 Q&A |
| 방식 | PPT / PDF / FreeCAD 라이브 시연 |
| 발표자 | 팀원 전원 발표 (1인 1파트) |
| 언어 | 자유 (권장: 한국어 + 영어 용어 혼용) |

### 발표 순서

```
1. 팀 소개 + 역할 분담 (1분)
   - "저는 모터 마운트 모델링과 FEA 해석을 담당했습니다"

2. 설계 과정 하이라이트 (2분)
   - 가장 어려웠던 점
   - 재미있었던 점
   - 설계 변경 사항

3. 해석 결과 (2분)
   - FEA 개선 전/후
   - 위상 최적화 결과
   - 운동학 시뮬레이션

4. 시연 (1분)
   - FreeCAD에서 Assembly4 애니메이션 실행
   - 또는 미리 녹화된 영상

5. 소감 + Q&A (1분)
   - "기억에 남는 한 가지"
```

## 6. 3일 전체 복습

### Day 1 — 모델링

```
FreeCAD 설치  →  2D 스케치  →  3D 형상  →  부품 완성
  (기초)       (구속 조건)   (Pad/Pocket)  (터틀봇 부품)
```

### Day 2 — 조립·해석·출력

```
A2plus 조립  →  FEA 해석  →  STL 출력  →  3D프린터
  (부품 결합)   (구조 검증)   (출력 준비)  (실물 제작)
```

### Day 3 — 해석 심화

```
운동학 분석  →  간섭+공차  →  동적 시뮬  →  위상 최적화  →  보고서
 (속도 예측)   (정밀 검증)   (움직임 확인)  (경량화)     (종합 정리)
```
