# Day 1 · Step 5 — 터틀봇 베이스 플레이트 모델링

**소요시간**: 1h 30min  
**목표**: 터틀봇의 몸체가 되는 베이스 플레이트(하판)를 모델링하고, 바퀴·모터·캐스터·전자부품 장착을 위한 구멍과 홈을 설계한다.

---

## 1. 베이스 플레이트 사양

| 항목 | 값 | 비고 |
|------|-----|------|
| 형상 | 원형 (디스크) | TurtleBot3 Burger 기준 |
| 직경 | 180mm | |
| 두께 | 5mm | 강성을 위한 최소 두께 |
| 재질 | PLA 또는 ABS (3D프린터 출력) | |
| 모터 장착 | 좌우 대칭, N20 모터 | 바퀴 중심선 정렬 |
| 볼 캐스터 | 1개, 전면 또는 후면 중앙 | Φ10mm 볼 |
| 배선 홀 | 중앙 Φ30mm | 케이블 정리용 |

---

## 2. 베이스 플레이트 모델링

### 2-1: 새 Body 생성

```
Create Body → 이름: "Base_Plate"
```

### 2-2: 베이스 형상 스케치 + Pad

```
XY_Plane → Create Sketch

1. Create Circle → 중심 (0,0) → 반지름 90mm
   → Distance: "90mm"
2. Pad → Length: "5mm" → OK
```

### 2-3: 모서리 Fillet

```
원형 판의 상/하단 모서리 → Fillet → R3mm
(날카로운 모서리를 둥글게 처리)
```

---

## 3. 모터 장착부 설계

터틀봇은 좌우에 N20 마이크로 DC 모터를 장착한다.

### 3-1: 모터 위치 결정

```
- 좌측 모터 중심: (-55, 0)
- 우측 모터 중심: (55, 0)
- 모터 축은 직경 6mm
- 모터 본체: 약 37mm × 15mm × 12mm
```

### 3-2: 모터 고정용 구멍

```
1. 베이스 플레이트 윗면 선택 → Create Sketch
2. 다음 구멍들을 스케치:

   [좌측 모터]
   - 모터 축 통과 구멍: Φ8mm (여유)
     → 중심 (-55, 0) → Distance: "4mm"
   - 고정 나사 구멍: Φ3mm (M3 볼트)
     → 위치: (-55, ±7.5)  (모터 폭 15mm 기준)
     → Distance: 반지름 "1.5mm"

   [우측 모터] (좌우 대칭)
   - 축 구멍: 중심 (55, 0) → Φ8mm → 반지름 "4mm"
   - 나사 구멍: (55, ±7.5) → Φ3mm → 반지름 "1.5mm"

3. Pocket → Through All → OK
```

> **팁**: 좌측 구멍을 다 그리고 조합보기에서 해당 스케치 선택 → `Mirror` → YZ_Plane (중앙 대칭면) → OK

---

## 4. 볼 캐스터 장착부

### 4-1: 볼 캐스터 위치

```
전면 중앙: (X=70, Y=0) — 전면에 배치하여 균형 유지
```

### 4-2: 캐스터 구멍

```
1. 윗면 → Create Sketch
2. 볼 캐스터 장착 나사 구멍 4개:
   - 중앙: (70, 0)
   - Φ3mm 구멍 4개 (M3 볼트, 사각형 배치)
   - 위치: (70±10, ±10)
3. Pocket → Through All
```

> 실제 볼 캐스터 규격에 맞게 조정 가능:
> - M3 나사 4개, 20mm × 20mm 정사각형 배치
> - 중앙 볼 직경 Φ10mm (통과 구멍은 Φ12mm로 여유)

---

## 5. 배선 홀 + 전자부품 공간

### 5-1: 중앙 배선 홀

모터 케이블과 센서 케이블이 아래로 통과할 원형 홀

```
1. 윗면 → Create Sketch
2. Create Circle → 중심 (0,0) → 반지름 15mm
3. Pocket → Through All
```

### 5-2: 전자부품 장착 홀 (선택)

Jetson Xavier NX 또는 배터리 팩 고정용 나사 구멍

```
- Jetson 보드 장착 간격 확인 후 나사 구멍 추가
- 예: 70mm × 50mm 사각형 모서리에 Φ3mm 구멍 4개
- 위치: (±35, ±25)
```

---

## 6. 완성 검토

### 조감도 (위에서 본 레이아웃)

```
                  ┌─ ① 볼 캐스터 (전면)
                  │
     ②좌측 모터──┼── ②우측 모터
                  │
                  │  ╭─────╮
                  │  │③배선│
                  │  │ 홀  │
                  ╰──╰─────╯
```

### 치수 확인

```
Base_Plate 전체:        Φ180mm × 두께 5mm
좌/우 모터 중심 간격:    110mm (55mm × 2)
캐스터 중심:            (70, 0)
배선 홀:                Φ30mm
나사 구멍:              Φ3mm (M3 볼트용, Through All)
```

---

## 7. 파일 저장

```
Ctrl+S → "turtlebot_base.FCStd"
```

---

## ❗ 설계 팁

| 상황 | 해결 방법 |
|------|----------|
| 구멍 위치가 헷갈린다 | 먼저 종이에 스케치하고 좌표 계산 |
| Pocket이 반대쪽에 생긴다 | Pocket → Direction → Reversed 체크 |
| 모터 규격이 다르다 | 실제 사용하는 모터의 데이터시트 확인 후 치수 조정 |
| 나사 구멍 크기 | M3 = Φ3mm (2.8~3.2mm 여유 있음) |
| Fillet이 적용 안 됨 | 먼저 Pocket 작업을 모두 완료한 후 Fillet 실행 |
