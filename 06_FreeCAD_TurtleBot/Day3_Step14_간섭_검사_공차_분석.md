**소요시간**: 1h 30min  
**목표**: A2plus로 전체 조립체의 간섭을 정밀 검사하고, 축-구멍 끼워맞춤 공차를 설계에 반영한다.

---

## 1. 간섭 검사 (Interference Detection)

### 1-1: 간섭이란?

> 간섭(Interference) = 두 부품이 동일한 공간을 차지하는 상태
> 실제로는 조립이 불가능하거나 부품이 파손됨

| 상태 | 의미 | 조치 |
|------|------|------|
| ✅ 간섭 없음 | 모든 부품이 독립 공간 점유 | 정상 |
| ⚠ 접촉 | 면이 정확히 맞닿음 | 의도적이면 OK |
| ❌ 간섭 | 부품이 겹침 | 설계 수정 필요 |

### 1-2: A2plus 간섭 검사 실행

```
1. turtlebot_assembly.FCStd 열기
2. 작업대 → A2plus
3. A2plus → Check Interference
   (또는 도구모음: 두 부품 겹친 아이콘)

4. 결과 창:
   ┌────────────────────────────────┐
   │ Interference Report            │
   ├────────────────────────────────┤
   │ Checking...                    │
   │ No interference detected ✅    │
   └────────────────────────────────┘
```

### 1-3: 간섭이 발견된 경우

```
⚠ Interference detected!
────────────────────────────────
Base_Plate ∩ Motor_Mount_L
  - Volume: 12.5 mm³
  - Location: (55, 0, 2.5)

→ 모터 마운트가 베이스 플레이트보다 아래로 내려간 상태
→ 해결: Motor_Mount_L의 Z축 위치 +2mm 조정
```

### 1-4: 간섭 수정 방법

```
1. 조합보기에서 Motor_Mount_L의 constraint 찾기
2. planeCoincident constraint 더블클릭
3. Offset 값을 조정 (예: 0 → 2mm)
4. OK → 재검사
```

## 3. 공차(Tolerance) 개념

### 3-1: 왜 공차가 필요한가?

> 3D프린터는 **완벽하게** 출력하지 않는다.

```
설계 치수: Φ6mm  →  실제 출력: Φ5.8~6.2mm (오차 ±0.2mm)
설계 치수: 50mm   →  실제 출력: 49.8~50.2mm
```

**공차**: 허용 가능한 치수 오차의 범위

### 3-2: 축-구멍 끼워맞춤 (Shaft-Hole Fit)

| 종류 | 설명 | 예제 | 사용처 |
|------|------|------|--------|
| **헐거움 (Clearance)** | 축이 구멍보다 작음 | Φ6mm 축 + Φ6.2mm 구멍 | 모터 축-바퀴 |
| **중간 (Transition)** | 거의 같은 크기 | Φ6mm 축 + Φ6.05mm 구멍 | 위치 결정 핀 |
| **죔 (Interference)** | 축이 구멍보다 큼 | Φ6mm 축 + Φ5.95mm 구멍 | 베어링 압입 |

> **터틀봇 적용**: 모터 축과 바퀴 구멍은 **헐거운 끼워맞춤** (조립/분해 용이)

### 3-3: 권장 공차 (3D프린터 기준)

| 상황 | 공차 | 예 |
|------|:----:|-----|
| 자유 회전 축-구멍 | +0.2mm | Φ6mm 축 → Φ6.2mm 구멍 |
| 볼트 체결 구멍 | +0.2mm | M3 볼트(Φ3) → Φ3.2mm 구멍 |
| 고정 위치 맞춤 | +0.1mm | 스탠드오프 삽입부 |
| 면 접촉 | ±0.0mm | 플레이트 간 맞닿음 |
| 나사산 | -0.1mm | M3 나사 구멍 → Φ2.8mm (탭핑용) |

## 4. FreeCAD로 공차 분석하기

### 4-1: 구멍 지름 측정

```
1. Part 워크벤치 전환
2. Measure → Measure Linear
3. 바퀴의 축 구멍 내면 두 점 선택 (원형 모서리)
   → 직경 = 6.2mm 확인
```

### 4-2: 공차 시뮬레이션 — 최악 조건(Worst Case) 분석

조립이 항상 성공하려면 **최악의 조건에서도** 조립 가능해야 한다.

```
최악 조건:
  축 최대: Φ6.0mm + 0.1mm (프린터 오차) = Φ6.1mm
  구멍 최소: Φ6.2mm - 0.1mm (프린터 오차) = Φ6.1mm
  
  → Φ6.1mm 축 + Φ6.1mm 구멍 = 간신히 조립 가능 ✅
```

### 4-3: 실습 — 구멍 크기별 조립 가능성

| 구멍 Φ | 축 Φ6mm +0.1mm | 조립 가능? |
|:-------:|:--------------:|:----------:|
| 6.0mm | 6.1mm | ❌ |
| 6.1mm | 6.1mm | ⚠ 끼움 |
| 6.2mm | 6.1mm | ✅ 여유 |
| 6.5mm | 6.1mm | ✅ 헐거움 |

> 3D프린터 출력물은 **Φ6.2mm 이상**의 구멍을 설계해야 안정적 조립 가능

## 5. 공차 적용 실습

### 5-1: 바퀴 축 구멍 수정 (필요 시)

```
1. turtlebot_wheel.FCStd 열기
2. Wheel_Body → Sketch (축 구멍 스케치) 더블클릭
3. Distance constraint: "3.1mm" (반지름 → Φ6.2mm)
4. 확인 → Body 재계산 (Ctrl+R)
```

### 5-2: 나사 구멍 일괄 확인

| 부품 | 구멍 용도 | 현재 Φ | 권장 Φ | 수정? |
|------|----------|:------:|:------:|:----:|
| Base_Plate | 모터 고정 M3 | 3.0mm | 3.2mm | 필요시 |
| Base_Plate | 캐스터 M3 | 3.0mm | 3.2mm | 필요시 |
| Motor_Mount | 모터 고정 M2 | 2.0mm | 2.2mm | 필요시 |
| Motor_Mount | 베이스 고정 M3 | 3.0mm | 3.2mm | 필요시 |
| Top_Plate | 스탠드오프 M3 | 3.0mm | 3.2mm | 필요시 |

### 5-3: 수정 방법

```
1. 해당 스케치 더블클릭
2. 구멍 원 선택 → Distance constraint 값 변경
3. (반지름 기준: 1.5mm → 1.6mm = Φ3.0mm → Φ3.2mm)
4. 확인
```

## 6. 공차 설계 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **회전부는 헐겁게** | 모터 축, 바퀴 등 회전하는 부분은 +0.2mm 여유 |
| **고정부는 정확하게** | 브라켓, 마운트 등 고정 부위는 +0.1mm |
| **볼트 구멍은 여유 있게** | M3 볼트용 구멍은 Φ3.2mm (조립 편의) |
| **3D프린터 오차 고려** | ±0.1~0.2mm 기본 가정 |
| **출력 방향 고려** | Z축 방향 치수가 가장 부정확 |
