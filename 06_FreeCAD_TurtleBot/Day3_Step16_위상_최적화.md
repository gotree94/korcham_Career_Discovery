# Day 3 · Step 16 — 위상 최적화 (Topology Optimization)

**소요시간**: 1h 30min  
**목표**: FEA 해석 결과를 바탕으로 응력이 적은 영역의 재료를 제거하여 부품을 경량화하고, 최적화 전후의 무게와 강도를 비교한다.

---

## 1. 위상 최적화(Topology Optimization)란?

> **위상 최적화**: 주어진 하중과 경계 조건 내에서 **재료를 가장 효율적으로 배치**하여 최소 무게로 최대 강도를 얻는 설계 방법

```
전통적 설계: "이쯤에 구멍을 내면 가벼워지겠지?" (직관)
위상 최적화: "컴퓨터야, 하중을 견디면서 가장 가벼운 형태를 찾아줘" (알고리즘)
```

### 실제 사용 사례

| 분야 | 예 |
|------|-----|
| 항공우주 | 보잉 777X 날개 구조 (30% 경량화) |
| 자동차 | Tesla 부품 위상 최적화 |
| 로봇 | 로봇 팔 링크 경량화 |
| 3D프린터 | Girod(조형물) 형태의 브라켓 |

> 위상 최적화 결과는 유기적이고 자연스러운 형태(뼈대 구조)로 나타난다.

---

## 2. 최적화 대상 선정

터틀봇 부품 중 경량화 효과가 큰 부품:

| 부품 | 현재 무게 (예상) | 최적화 목표 | 비고 |
|------|:--------------:|:----------:|------|
| **모터 마운트** | ~8g | 4g (50% 감량) | 가장 효과 큼 |
| LiDAR 마운트 | ~10g | 6g | 큰 하중 없음 |
| **베이스 플레이트** | ~80g | 60g | 배선 홀 확장 |
| 볼 캐스터 브라켓 | ~6g | 4g | 단순 부품 |

> **1순위: 모터 마운트** — 작은 부품이지만 여러 개 사용되고, 하중 조건이 명확함

---

## 3. FreeCAD FEM 기반 최적화 방법

FreeCAD는 전용 위상 최적화 도구가 없지만, **FEM 해석 결과를 보고 수동으로 재료 제거**하는 방식으로 접근한다.

### 워크플로우

```
1. FEM 해석 실행 (Step 9 복습)
2. 응력 분포 확인 (낮은 응력 영역 식별)
3. 낮은 응력 영역에 Pocket 추가 (재료 제거)
4. 재해석하여 강도 확인
5. 2~4 반복 → 최적 형상 도출
```

---

## 4. 실습 — 모터 마운트 경량화

### 4-1: 현재 상태 분석

```
1. turtlebot_mounts.FCStd → Motor_Mount_L Body
2. FEM 해석 실행 (Step 9 참조)
3. Von Mises Stress 결과 확인

   현재 무게: 8g (예상)
   최대 응력: 11MPa (Step 9 개선 후 기준)
   안전율: 3.64
   → 안전율이 2.0 이상이므로 재료 제거 가능!
```

### 4-2: 저응력 영역 식별

FEA 결과에서 **파란색(저응력) 영역**이 재료 제거 대상이다.

```
FEA 색상 맵:
  빨강 (11MPa) → 고응력   [보존]
  노랑 (5MPa)  → 중간     [부분 보존]
  초록 (2MPa)  → 저응력   [제거 가능]
  파랑 (0MPa)  → 최저     [제거 대상]
```

### 4-3: 재료 제거 (Pocket 추가)

```
1. Motor_Mount_L Body 편집 (더블클릭)
2. 저응력 영역의 면 선택 → Create Sketch
3. 다음 위치에 재료 제거용 스케치:

   [제거 영역 1 — 중앙부]
   - 사각형: 10mm × 6mm
   - 위치: 모터 마운트 중앙 (응력 낮은 부분)
   - Pocket → Depth: "3mm" (완전 관통하지 않음)

   [제거 영역 2 — 모서리]
   - 원: Φ4mm
   - 위치: 응력 분포도에서 파란색 영역
   - Pocket → Through All (가능하면)

   [제거 영역 3 — 추가]
   - 타원 또는 사각형
   - 남은 파란 영역 모두 제거
```

### 4-4: 핵심 응력 경로 보존

**절대 제거하지 말아야 할 영역**:

```
┌─────────────────────────┐
│  🟡구멍    ← 🟡나사 구멍            │
│     \         /         │
│      🟡하중 경로 🟡    ← 힘이 전달되는 길
│       \     /           │
│        🟡고정🟡        ← 베이스 고정부
└─────────────────────────┘

→ 빨강/노랑 영역은 하중 경로(Load Path) — 절대 제거 금지!
```

> **하중 경로(Load Path)**: 힘이 전달되는 연속적인 경로. 이 경로를 끊으면 부품이 파손된다.

---

## 5. 최적화 반복 (Iteration)

### 1차 최적화

```
재료 제거: 중앙부 3mm 깊이 포켓
재해석 결과:
  - 무게: 8g → 7g (-12.5%)
  - 최대 응력: 11MPa → 13MPa (↑ 증가했지만 안전)
  - 안전율: 3.64 → 3.08 (여전히 2.0 이상 ✅)
```

### 2차 최적화

```
재료 제거: 모서리 2개 Φ4mm 관통
재해석 결과:
  - 무게: 7g → 6.2g (-22.5%)
  - 최대 응력: 13MPa → 16MPa
  - 안전율: 3.08 → 2.50 (여전히 OK ✅)
```

### 3차 최적화 (최종)

```
재료 제거: 추가 타원 포켓
재해석 결과:
  - 무게: 6.2g → 5.5g (-31.3%)
  - 최대 응력: 16MPa → 18MPa
  - 안전율: 2.50 → 2.22 (✅ 안전 기준 충족)
```

---

## 6. 최적화 전후 비교표

### 모터 마운트

| 항목 | 최적화 전 | 최적화 후 | 개선율 |
|------|:---------:|:---------:|:------:|
| 무게 | 8.0g | 5.5g | **-31%** |
| 최대 응력 | 11MPa | 18MPa | +64% |
| 안전율 | 3.64 | 2.22 | -39% |
| 재료 부피 | 7,620mm³ | 5,240mm³ | **-31%** |
| 출력 시간 | 90분 | 65분 | **-28%** |

### 베이스 플레이트 (확장 실습)

| 항목 | 최적화 전 | 최적화 후 | 개선율 |
|------|:---------:|:---------:|:------:|
| 무게 | 80g | 62g | **-22.5%** |
| 안전율 | >10 | 8.5 | 충분 |

> **경량화 효과**: 필라멘트 절약 → 비용 절감, 로봇 자체 무게 감소 → 배터리 효율 ↑

## 7. 최적 형상 STL 출력

### 7-1: 최종 최적화 부품 내보내기

```
1. Motor_Mount_L Body 선택
2. File → Export → STL
   → "motor_mount_optimized.stl"
3. Motor_Mount_R도 동일하게 최적화 후 내보내기
   (Mirror로 대칭 설계했다면 동일한 최적화 적용 가능)
```

### 7-2: 최적화 전후 STL 비교

```
motor_mount_original.stl      → 8g 예상
motor_mount_optimized.stl     → 5.5g 예상
```

---

## 8. 위상 최적화 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **하중 경로를 보존하라** | 응력이 집중된 부분은 절대 제거 금지 |
| **반복하라** | 한 번에 많이 제거하지 말고, 조금씩 제거하며 확인 |
| **안전율 2.0 유지** | 최종 안전율이 2.0 이상인지 항상 확인 |
| **대칭 활용** | 좌우 대칭 부품은 한쪽만 최적화 후 Mirror |
| **3D프린터 고려** | 너무 얇은 부분( < 1mm )은 출력 불가 |
