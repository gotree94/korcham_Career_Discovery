# Day 3 · Step 13 — 운동학(Kinematics) 해석

**소요시간**: 1h  
**목표**: 터틀봇 바퀴의 회전 운동을 선형 속도로 변환하고, 바퀴 간 속도 관계를 분석하여 로봇의 이동 방향을 예측한다.

---

## 1. 운동학(Kinematics)이란?

> **운동학**: 힘의 원인을 고려하지 않고 물체의 **위치·속도·가속도** 변화만을 다루는 학문

```
로봇 운동학 = 바퀴 회전 속도 → 로봇의 이동 속도/방향 변환
```

### 터틀봇에 적용

터틀봇은 **차동 구동(Differential Drive)** 방식:
- 좌측 바퀴 속도 ≠ 우측 바퀴 속도 → 회전
- 좌측 바퀴 속도 = 우측 바퀴 속도 → 직진
- 좌측 바퀴 속도 = -우측 바퀴 속도 → 제자리 회전

---

## 2. 바퀴 운동학 기본 공식

### 2-1: 회전 각속도 → 선속도

```
v = r × ω

v: 바퀴 중심의 선속도 (mm/s)
r: 바퀴 반지름 (mm)
ω: 바퀴 회전 각속도 (rad/s)
```

#### 예제: 터틀봇 바퀴

```
바퀴 직경: 66mm  →  r = 33mm
모터 100RPM      →  ω = 100 × 2π / 60 = 10.47 rad/s

v = 33mm × 10.47 rad/s = 345.5 mm/s  ≈ 0.35 m/s
```

### 2-2: 차동 구동 속도 계산

```
V = (V_right + V_left) / 2          (로봇 전체 선속도)
Ω = (V_right - V_left) / W          (로봇 회전 각속도)

V_right: 우측 바퀴 선속도
V_left:  좌측 바퀴 선속도
W:       좌우 바퀴 사이 거리 (휠베이스)
```

#### 예제: 터틀봇 회전

```
휠베이스 W = 110mm (베이스 플레이트 모터 간격)

좌측 바퀴: 100 RPM → v_left = 345.5 mm/s
우측 바퀴: 50 RPM  → v_right = 172.75 mm/s

V = (172.75 + 345.5) / 2 = 259.1 mm/s
Ω = (172.75 - 345.5) / 110 = -1.57 rad/s (≈ -90°/s, 우회전)

→ 로봇은 오른쪽으로 회전하며 전진
```

## 3. FreeCAD로 운동학 검증

### 3-1: 바퀴 위치 측정

```
1. turtlebot_assembly.FCStd 열기
2. 작업대 → "Part" 전환
3. Measure → Measure Linear
   - 좌측 바퀴 중심 선택
   - 우측 바퀴 중심 선택
   → 거리 = 110mm 확인
```

### 3-2: 회전 반경 계산

```
R = W / 2 × (v_right + v_left) / (v_right - v_left)

위 예제:
R = 110/2 × (172.75 + 345.5) / (172.75 - 345.5)
  = 55 × 518.25 / (-172.75)
  = -165mm

→ 로봇은 중심으로부터 165mm 반경으로 우회전
```

### 3-3: FreeCAD에 경로 그리기

```
1. Draft 워크벤치 전환
2. Draft → Arc → 중심 (0,0) → 반지름 165mm → 각도 0~90°
3. 회전 경로 시각화 완료
```

## 4. 실습 — 엑셀/스프레드시트 운동학 계산기

### 입력값

```
바퀴 직경 (mm):   66
좌측 모터 RPM:    100
우측 모터 RPM:    50
휠베이스 (mm):    110
```

### 계산 결과

| 항목 | 값 | 단위 |
|------|-----|:----:|
| 바퀴 반지름 r | 33.0 | mm |
| 좌측 각속도 ω_L | 10.47 | rad/s |
| 우측 각속도 ω_R | 5.24 | rad/s |
| 좌측 선속도 v_L | 345.5 | mm/s |
| 우측 선속도 v_R | 172.8 | mm/s |
| 로봇 선속도 V | 259.1 | mm/s |
| 로봇 각속도 Ω | -1.57 | rad/s |
| 회전 반경 R | -165.0 | mm |
| 방향 | 우회전 전진 | |

### 실습 과제

다음 4가지 주행 패턴의 운동학을 계산하라:

| 패턴 | 좌측 RPM | 우측 RPM | 예상 이동 |
|------|:--------:|:--------:|----------|
| 직진 | 100 | 100 | 전진 |
| 우회전 | 100 | 50 | 우측 선회 |
| 제자리 회전 | 100 | -100 | 제자리 좌회전 |
| 급회전 | 50 | -50 | 제자리 우회전 |

> 각 패턴의 V, Ω, R을 계산하고 FreeCAD Draft로 경로를 그려보자.

## 5. FreeCAD Measure 도구 익히기

| 도구 | 기능 | 단축키 |
|------|------|:------:|
| Measure Linear | 두 점/모서리 사이 거리 | Shift+M |
| Measure Angle | 두 선분 사이 각도 | Shift+A |
| Measure Clearance | 면 사이 최소 거리 | Shift+C |

### 실습: 조립체 측정

```
1. 조립 파일에서 Base_Plate 윗면 선택
2. Top_Plate 밑면 선택
3. Measure Clearance → 30.0mm (스탠드오프 높이)
```

---

## 6. 핵심 요약

```
바퀴 RPM → 각속도 ω → 선속도 v → 로봇 속도 V, Ω

   ω = RPM × 2π / 60
   v = r × ω
   V = (v_R + v_L) / 2
   Ω = (v_R - v_L) / W
```

> **운동학 해석 = 로봇이 어떻게 움직일지 숫자로 예측하는 작업**
