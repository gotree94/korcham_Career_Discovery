# Day 2 · Step 12 — 설계 리뷰 + 팀 발표

**소요시간**: 30min  
**목표**: 2일간의 기구 설계 과정을 팀별로 정리하고 발표하며, 설계 결과물을 공유한다.

---

## 1. 팀별 체크리스트

### 완성된 파일 목록 확인

```
□ turtlebot_wheel.FCStd          — 바퀴 + 타이어
□ turtlebot_base.FCStd           — 베이스 플레이트
□ turtlebot_mounts.FCStd         — 모터 마운트 + 캐스터
□ turtlebot_top.FCStd            — 상판 (선택)
□ turtlebot_lidar.FCStd          — LiDAR 마운트 (선택)
□ turtlebot_assembly.FCStd       — 전체 조립
□ turtlebot_assembly.step        — STEP 교환 형식
□ *.stl (8개 부품)               — STL 출력 파일
□ fea_before.png / fea_after.png — FEA 결과 스크린샷
```

### 조립 확인

- [ ] 볼 캐스터가 정확히 조립되었는가?
- [ ] 좌/우 바퀴가 회전 간섭 없이 조립되었는가?
- [ ] 상판-하판 스탠드오프가 정확히 정렬되었는가?
- [ ] LiDAR 마운트가 수평으로 장착되었는가?

---

## 2. 팀 발표 준비 (10분)

### 발표 내용 (1팀당 5분)

```
슬라이드 1: 팀 소개 + 역할 분담
  - 모델링 담당, 조립 담당, FEA 담당, 출력 담당

슬라이드 2: 설계 과정
  - 가장 어려웠던 부품과 해결 방법
  - 설계 중 변경한 사항과 이유

슬라이드 3: FEA 결과
  - 개선 전/후 비교 이미지
  - 안전율 변화

슬라이드 4: 출력 결과 (선택)
  - 3D프린터 출력물 사진
  - 실제 조립 사진

슬라이드 5: 소감
  - "기억에 남는 한 가지"
  - 앞으로 더 배우고 싶은 것
```

### 발표 양식

```
자유 형식 (종이 포스터, PPT, FreeCAD 직접 시연 모두 가능)
PC 또는 모바일 화면 공유
```

## 3. 평가 기준

| 항목 | 배점 | 평가 내용 |
|------|:----:|----------|
| 모델링 완성도 | 30% | 모든 부품이 정확한 치수로 모델링되었는가? |
| 조립 정확도 | 20% | 간섭 없이 모든 부품이 결합되었는가? |
| FEA 해석 | 20% | 해석이 정상 실행되고 개선 전후 비교가 있는가? |
| 출력 준비 | 15% | STL 파일이 정상 생성되고 슬라이싱 설정이 적절한가? |
| 발표 | 15% | 과정과 결과를 효과적으로 전달했는가? |

## 4. 전체 과정 복습

### 2일간의 여정

```
Day 1: 3D 모델링 기초
  ┣━ FreeCAD 포터블 설치
  ┣━ 2D 스케치 (선/원/사각형 + 구속 조건)
  ┣━ 3D 형상 (Pad/Pocket/Revolution)
  ┣━ 터틀봇 바퀴 + 타이어
  ┣━ 베이스 플레이트 (모터/캐스터/배선 홀)
  ┗━ 모터 마운트 + 볼 캐스터 브라켓

Day 2: 조립 + 해석 + 출력
  ┣━ A2plus 조립 (전체 터틀봇)
  ┣━ LiDAR 마운트 + 상판
  ┣━ FEA 구조 해석 (설계 개선 전후 비교)
  ┣━ STL 출력 + 슬라이싱
  ┣━ 3D프린터 출력 시연
  ┗━ 최종 리뷰 + 발표
```

### 핵심 키워드

```
✅ Fully Constrained  — 스케치의 모든 자유도 고정
✅ Pad / Pocket       — 2D→3D 돌출 / 컷
✅ Revolution         — 축 중심 회전
✅ PlaneCoincident    — A2plus 면 정렬 조립
✅ CircularEdge       — A2plus 축 정렬 조립
✅ Von Mises Stress   — FEA 응력 해석 결과
✅ STL                — 3D프린터 표준 출력 형식
✅ G-code             — 프린터 동작 명령어
```

## 5. 기억에 남는 한 가지

> 각자 **"이번 기구 설계 과정에서 가장 인상 깊었던 것"** 을 한 문장으로 적어서 공유한다.

예시:
- "내가 그린 2D 스케치가 순식간에 3D 입체가 되는 게 신기했다."
- "빨간색(FEA 응력 집중)이 초록색으로 바뀌는 걸 직접 봤다."
- "A2plus에서 부품을 조립할 때 딱 맞아떨어지는 느낌이 쾌감이었다."
- "3D프린터가 출력하는 모습을 보며 내 설계가 실제 물건이 되는 걸 체험했다."
