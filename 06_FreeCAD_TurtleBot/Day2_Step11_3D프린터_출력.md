# Day 2 · Step 11 — 3D프린터 출력 시연

**소요시간**: 1h  
**목표**: 3D프린터의 실제 출력 과정을 시연하고, 출력물을 직접 확인하며 기구 설계의 완성품을 경험한다.

---

## 1. 3D프린터 기본 구조

```
         ┌───── Z축 (위/아래) ─────┐
         │                         │
     ╔═══╧═══════════════════════╧═══╗
     ║        필라멘트 스풀          ║
     ╚═══╤═══════════════════════╤═══╝
         │                         │
         │   ┌─────────────────┐   │
         │   │  익스트루더     │   │
         │   │  (노즐 0.4mm)   │   │
         │   └────────┬────────┘   │
         │            │            │
         │   ┌────────┴────────┐   │
         │   │    출력물       │   │
         │   │  ┌──────────┐   │   │
    ═════╪═══╪══╪═ ═ ═ ═ ═ ═╪═══╪═══ X축 (좌/우)
         │   │  └──────────┘   │   │
         │   └─────────────────┘   │
         │      베드 (Y축 전/후)   │
         └─────────────────────────┘
```

| 구성 요소 | 설명 |
|-----------|------|
| **익스트루더** | 필라멘트를 녹여서 노즐로 토출 |
| **노즐** | 0.4mm 구멍으로 용융 필라멘트 압출 |
| **베드** | 출력물이 붙는 플랫폼 (가열 가능) |
| **필라멘트** | 1.75mm 직경의 플라스틱 선 |

---

## 2. 출력 전 준비

### 2-1: 프린터 상태 확인

- [ ] 베드가 수평(레벨링)인가?
- [ ] 노즐이 막히지 않았는가?
- [ ] 필라멘트가 충분한가? (~50g 이상)
- [ ] SD카드에 G-code가 저장되었는가?

### 2-2: 필라멘트 장착

```
1. 필라멘트 스풀을 거치대에 장착
2. 필라멘트 끝을 약 45° 로 잘라냄
3. 익스트루더의 필라멘트 입구에 삽입
4. 프린터 메뉴: Prepare → Preheat PLA (또는 ABS)
   - PLA: 노즐 200°C, 베드 60°C
5. 익스트루더 손으로 밀어 넣기 또는 메뉴에서 Load Filament
6. 노즐에서 필라멘트가 나오는지 확인
```

### 2-3: 베드 레벨링

```
1. 프린터 메뉴: Prepare → Auto Home (모든 축 원점 복귀)
2. Prepare → Bed Leveling
3. 종이 한 장으로 노즐과 베드 사이 간격 확인:
   - 종이가 살짝 걸리면 OK
   - 너무 타이트하면 Z축 올림
   - 너무 느슨하면 Z축 내림
4. 베드 4개 모서리 + 중앙 총 5포인트 확인
```

---

## 3. 출력 실행

### 3-1: G-code 로드

```
방법 ① SD카드:
  G-code 파일을 SD카드에 복사
  프린터 메뉴: Print from SD → 파일 선택

방법 ② USB 연결:
  Cura → Monitor → USB로 연결
  "Print via USB" 버튼
```

### 3-2: 출력 과정 관찰

```
⏱ 타임라인:

[00:00] Auto Home → 베드/노즐 가열 시작
[05:00] 목표 온도 도달 → 출력 시작
[05:10] Skirt/Brim 출력 (첫 레이어 준비)
[06:00] 첫 레이어 출력 시작 (가장 중요!)
[10:00] 첫 레이어 완료 → 본격 적층
[나머지] 계속 적층...
[종료] 출력 완료 → 베드 냉각 → 출력물 분리
```

### 3-3: 관찰 포인트

| 시간 | 관찰할 점 |
|------|----------|
| 첫 레이어 | 재료가 베드에 잘 붙는가? (밀착도 확인) |
| 10분 후 | 모서리가 들리지 않는가? |
| 30분 후 | 레이어 간 결합이 균일한가? |
| 중간 | Stringing(거미줄)이 생기지 않는가? |
| 종료 전 | 상단면이 평탄한가? |

---

## 4. 출력 후 처리

### 4-1: 출력물 분리

```
1. 베드가 식을 때까지 대기 (베드 40°C 이하)
2. 스크래퍼(주걱)를 조심히 밀어 넣어 분리
3. ⚠ 노즐과 베드 화상 주의!
```

### 4-2: 서포트 제거

```
1. 니퍼/플라이어로 서포트 조심히 제거
2. 사포(#400~#600)로 서포트 자국 제거
3. 구멍 내부 서포트는 드릴 비트 or 공구로 제거
```

### 4-3: 후처리

| 작업 | 도구 | 설명 |
|------|------|------|
| 버 제거 | 사포 / 파일 | 날카로운 모서리 다듬기 |
| 구멍 청소 | 핀 드릴 | Φ3mm 구멍에 볼트 들어가는지 확인 |
| 표면 처리 | 아세톤 증기 (ABS만) | 표면을 녹여 매끄럽게 (선택) |
| 조립 테스트 | M3 볼트+너트 | 구멍에 볼트가 잘 체결되는지 확인 |

---

## 5. 출력물 품질 평가

### 5-1: 체크리스트

- [ ] 치수가 설계와 일치하는가? (캘리퍼스 측정)
- [ ] 구멍이 너무 작거나 크지 않은가?
- [ ] 표면이 매끄러운가?
- [ ] 레이어 간 분리가 없는가?
- [ ] 워핑(휨)이 없는가?

### 5-2: 문제해결

| 문제 | 원인 | 해결 |
|------|------|------|
| 워핑 (모서리 들림) | 베드 온도 낮음 | Brim 추가, 베드 온도 ↑ |
| 레이어 분리 | 출력 온도 낮음 | 온도 5~10°C ↑ |
| 구멍 작음 | 수축 보상 없음 | Cura에서 Hole Horizontal Expansion +0.2mm |
| 표면 거침 | 레이어 높이 큼 | 0.2mm → 0.12mm |
| 실패한 출력 | 베드 레벨링 불량 | 재레벨링 후 재시도 |

---

## 6. 시연 실습 — 강사 시연 + 조별 체험

### 강사 시연 (20분)

```
1. 사전 출력된 샘플 부품 전시
2. G-code 로드 → 출력 시작 과정 시연
3. 첫 레이어 출력 관찰 (베드 밀착 확인)
4. 향후 2~4시간 동안 자동 출력 설명
```

### 조별 체험 (40분)

```
1. 각 조별로 1개 부품 STL 선택
2. Cura에서 슬라이싱 설정 직접 구성
3. G-code 생성
4. (시간 관계상 실제 출력은 강사가 대행)
5. 강사가 사전 출력한 부품으로 조립 체험
```

- [ ] 필라멘트 장착과 베드 레벨링을 체험했다
- [ ] G-code 로드부터 첫 레이어 출력까지 관찰했다

- [ ] 출력물의 치수 정확도를 평가했다
- [ ] 사전 출력된 부품으로 실제 조립을 테스트했다
