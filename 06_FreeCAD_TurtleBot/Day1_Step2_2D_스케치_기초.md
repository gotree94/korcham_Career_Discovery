# Day 1 · Step 2 — 2D 스케치 완전 정복

**소요시간**: 1h 20min  
**목표**: 스케치 평면 선택, 기본 도형 그리기, 치수/기하 구속 조건을 익히고 완전 구속된 스케치를 작성한다.

---

## 1. 스케치(Sketch)란?

3D 모델링의 출발점은 **2D 스케치**다. 종이에 물체의 단면을 그리고, 그 단면을 입체로 만드는 방식이다.

> 스케치(Sketch) → Pad(돌출) → 3D 형상
>
> 자동차를 만들기 전에 설계도부터 그리는 것과 같다.

---

## 2. 새 문서 + 스케치 시작

### 2.1 새 문서

```
File → New    (단축키: Ctrl+N)
```

### 2.2 Part Design 작업대로 전환

작업대 콤보박스 → **Part Design** 선택

### 2.3 스케치 생성

```
Part Design → Create Sketch (도구모음 버튼)
```

**평면 선택 대화상자**가 나타난다:

```
Select a plane for the sketch:
┌─────────────────────────────────────┐
│ ○  XY_Plane (위에서 본 평면 — 평면도)│
│ ○  XZ_Plane (정면 — 정면도)         │
│ ○  YZ_Plane (측면 — 측면도)         │
│ ○  (선택한 면)                      │
└─────────────────────────────────────┘
```

> **첫 실습**: XY_Plane 선택 → OK

---

## 3. 기본 도형 그리기

Sketcher 작업대가 활성화된다. 왼쪽 도구모음에 스케치 도구들이 나타난다.

### 3.1 선(Line) 그리기

```
Sketcher → Create Line (단축키: L)
```

1. 3D 뷰에서 **첫 점 클릭**
2. **끝 점 클릭**
3. 우클릭 또는 `Esc`로 종료

> **실습**: 가로선 1개, 세로선 1개를 그려보자.

### 3.2 사각형(Rectangle) 그리기

```
Sketcher → Create Rectangle (단축키: R)
```

1. 첫 모서리 클릭
2. 대각선 반대 모서리 클릭

> **실습**: 임의의 사각형을 3개 그려보자.

### 3.3 원(Circle) 그리기

```
Sketcher → Create Circle (단축키: C)
```

1. 중심점 클릭
2. 반지름 방향 클릭 (크기 지정)

> **실습**: 사각형 안에 원을 하나 그려보자.

### 3.4 호(Arc) 그리기

```
Sketcher → Create Arc → 3 points
```

세 점(시작점, 끝점, 호 위의 점)을 차례로 클릭

## 4. 구속 조건(Constraints) — 정확한 도면의 핵심

스케치 도형은 그냥 그리면 위치와 크기가 정해지지 않는다. **구속 조건**을 추가하여 정확하게 고정한다.

### 4.1 구속 조건의 종류

| 종류 | 내용 | 아이콘 |
|------|------|:---:|
| **수평 구속 (Horizontal)** | 선분을 수평으로 고정 | `─` |
| **수직 구속 (Vertical)** | 선분을 수직으로 고정 | `│` |
| **거리 구속 (Distance)** | 선분 길이 / 점 사이 거리 고정 | `⇔` |
| **각도 구속 (Angle)** | 두 선분 사이 각도 고정 | `∠` |
| **동심 구속 (Coincident)** | 두 점을 같은 위치에 고정 | `⊙` |
| **접선 구속 (Tangent)** | 선과 원을 접하도록 고정 | `⌒` |
| **동일 구속 (Equal)** | 두 선분 길이를 같게 | `＝` |
| **대칭 구속 (Symmetric)** | 중심선 기준 대칭 | `↕` |
| **고정 구속 (Lock)** | 점의 좌표를 고정 | `🔒` |

### 4.2 구속 추가 방법

```
구속할 요소(선/점/원) 선택 → 구속 조건 버튼 클릭
```

#### 예: 사각형을 정확히 50mm × 30mm 로 만들기

```
1. 사각형의 한 변 선택
2. Distance 버튼 클릭
3. 입력창에 "50mm" 입력 → Enter
4. 인접 변 선택
5. Distance 버튼 → "30mm" 입력
```

## 5. 완전 구속(Fully Constrained)

### 5.1 완전 구속이란?

스케치에 있는 모든 도형의 위치와 크기가 수치로 완전히 결정된 상태를 말한다.

| 상태 | 조합보기 표시 | 의미 |
|------|:---:|------|
| **완전 구속** | ✅ (초록) | 더 이상 추가 정보가 필요 없음 |
| **미구속** | ❌ (노란 경고) | 아직 위치/크기가 정해지지 않음 |
| **과잉 구속** | ⛔ (빨강) | 중복된 구속 — 일부 제거 필요 |

### 5.2 실습: 완전 구속된 직사각형

```
1. Create Rectangle (XY 평면)
2. 사각형 아래 변 선택 → 수평(Horizontal) 구속
3. 사각형 오른쪽 변 선택 → 수직(Vertical) 구속
4. 아래 변 선택 → Distance → "100mm"
5. 오른쪽 변 선택 → Distance → "60mm"
6. 왼쪽 아래 꼭지점 선택 → Lock (원점에 고정)

→ 조합보기에서 "✅ Fully constrained" 확인
```

## 6. 실습 과제 — 터틀봇 바퀴 스케치 준비

터틀봇 구동바퀴의 측면 형상을 스케치해보자.

**완성 목표**: Φ66mm 원 (바퀴 외곽) + Φ6mm 원 (축 구멍) + 키 홈

### 따라 하기

```
1. File → New → Part Design → Create Sketch (XY_Plane)
2. Create Circle → 중심 (0,0) → 반지름 33mm
3. 반지름 선택 → Distance → "33mm" 입력
4. Create Circle → 중심 (0,0) → 반지름 3mm (축 구멍)
5. 반지름 선택 → Distance → "3mm" 입력
6. 결과 확인: "✅ Fully constrained (2 degrees of freedom)"
```

> 이 스케치는 다음 Step에서 Pad(돌출)하여 3D 바퀴가 된다.

---

## 7. 스케치 꿀팁

| 팁 | 설명 |
|----|------|
| **그리드 스냅** | 도구모음 토글 버튼으로 ON/OFF, 초보자는 ON 권장 |
| **자동 구속** | Preferences → Sketcher → Auto constraints 활성화 |
| **치수 단위** | 기본 mm, "1m" 또는 "5cm" 도 입력 가능 |
| **Undo** | `Ctrl+Z`로 실수 복구 |
| **구속 색상** | 초록=완전구속, 파랑=미구속, 빨강=과잉구속 |
| **Constrain режим** | 치수 입력 후 Enter 누르면 확정 |
