## 3. 감속기 선택 기준

### 3.1 감속기 유형 비교

| 항목 | 하모닉 드라이브 | 사이클로이드 감속기 |
|------|----------------|---------------------|
| 감속비 범위 | 30:1 ~ 160:1 | 6:1 ~ 119:1 |
| 정밀도 | 매우 높음 (1arcmin 이하) | 높음 (1~2arcmin) |
| 효율 | 85~90% | 90~95% |
| 충격 하중 내성 | 중간 | 높음 |
| 크기 | 컴팩트 | 상대적 큼 |
| 비용 | 높음 | 중간 |
| 주요 응용 | 로봇, 반도체 장비 | 산업용 로봇, CNC |

### 3.2 선택 근거

**하모닉 드라이브 적용 축 (1, 2, 4, 6축):**
- 높은 감속비 필요
- 컴팩트한 크기 요구
- 정밀한 위치 제어 필요
- 낮은 관성 모멘트 요구

**사이클로이드 감속기 적용 축 (3, 5축):**
- 높은 충격 하중 내성 필요
- 높은 효율 요구
- 비용 효율성 고려
- 중간 감속비 요구
